## 문제 9

```java
static interface F {
    int apply(int x) throws Exception;
}

static int run(F f) {
    try {
        return f.apply(2);
    } catch (Exception e) {
        return 10;
    }
}

public static void main(String[] args) {
    F f = (x) -> {
        if (x > 2) throw new Exception();
        return x * 2;
    };
    System.out.print(run(f) + run((int n) -> n + 9));
}
```
