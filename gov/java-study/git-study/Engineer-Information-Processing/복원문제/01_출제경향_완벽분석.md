### 1-3. 핵심 인사이트

- **프로그래밍이 왕이다:** 전체 180문제 중 77문제(42.8%)가 코드 문제. 코드를 못 읽으면 합격 불가.
- **C언어가 최다 출제:** Java보다 C가 매 회차 꾸준히 더 많이 나옴 (36 vs 31).
- **DB는 반드시 3~4문제:** 한 번도 빠진 적 없는 고정 과목.
- **운영체제는 0~2문제:** 나오면 1문제, 최근 들어 2문제까지 증가 추세.
- **데이터통신은 간헐적:** 9회 중 5회만 출제, 나올 때 1~2문제.

### 2-2. 주요 트렌드 변화 포인트

#### (1) Java 출제 비중 확대
- 2023년: 평균 3.0문제/회 -> 2024~2025년: 평균 3.7문제/회
- **상속, 다형성, 예외처리** 관련 문제가 특히 증가
- 2025년부터 **람다식, enum, 제네릭** 등 고급 문법 등장

#### (2) 운영체제 출제 증가
- 2023년: 3회 합쳐 1문제 -> 2024~2025년: 매년 3문제
- **페이지 교체 알고리즘(LRU/LFU)** 이 2024-1, 2024-3 연속 출제
- **CPU 스케줄링(SJF, SRT, RR)** 이 2024-2, 2025-2 연속 출제
- 운영체제가 "나오면 좋겠다" 수준에서 **"반드시 대비"** 수준으로 격상

#### (3) 소프트웨어공학 심화
- 디자인 패턴: Proxy, Singleton, Visitor, Iterator, Abstract Factory, Adapter, 행위패턴 등 **매우 다양하게 출제**
- 커버리지: 분기, 조건, MC/DC, 문장 커버리지가 **거의 매 회차 출제**
- 결합도/응집도: 2024년부터 구체적 유형 묻는 문제 증가

#### (4) 네트워크 출제 감소
- 2023년 7문제 -> 2025년 5문제로 점진적 감소
- 단, **서브넷 계산**이 2025년 1회, 2회 연속 출제 (계산 문제 비중 증가)
- 순수 프로토콜 암기(RIP, OSPF 등)보다 **실무적 개념** 중심으로 변화

#### (5) 새로 등장한 토픽들
| 시기 | 새로 등장한 주제 |
|------|-----------------|
| 2024년 | MC/DC 커버리지, APT 공격, 제3정규형 심화 |
| 2025년 | 세션 하이재킹, 스캐어웨어, OTP, 비트연산 구조체 |
| 2025-2~3 | Java 람다식, Java enum, 라운드로빈 스케줄링 |

#### (6) 퇴조하는 토픽들
- **HDLC 프레임**: 2023-2 이후 미출제
- **ATM**: 2023-3 이후 미출제
- **순수 암기형 보안 용어**: 개념 설명형에서 **공격 시나리오 이해형**으로 변화

### 3-1. C언어 출제 분석 (총 36문제)

#### 세부 유형별 분류

| 유형 | 문항 수 | 출제 회차 | 중요도 |
|------|:-------:|----------|:------:|
| **포인터/문자열 포인터** | 7 | 23-1, 23-2, 23-3, 24-2, 24-3, 25-2(x2) | ★★★★★ |
| **구조체 (포인터 포함)** | 6 | 23-3, 24-1, 24-2, 25-1, 25-2, 25-3 | ★★★★★ |
| **연결리스트** | 5 | 24-2, 24-3, 25-1, 25-2, 25-3 | ★★★★★ |
| **문자열 처리** | 4 | 23-1, 24-1, 24-2, 25-3 | ★★★★ |
| **배열** | 3 | 23-2, 24-2, 25-1 | ★★★ |

| **switch-case** | 2 | 23-2, 24-2 | ★★ |
| **스택/큐 (자료구조)** | 2 | 23-2, 25-2 | ★★★ |
| **조건/삼항 연산자** | 2 | 24-1, 25-3 | ★★ |
| **static 변수** | 1 | 24-3 | ★★ |

#### C언어 출제 트렌드 핵심

1. **포인터는 불변의 1순위**: 매 회차 최소 1문제, 특히 **문자열 포인터** + **구조체 포인터** 조합이 킬러
2. **연결리스트 폭증**: 2024년부터 매 회차 출제. **노드 삽입/삭제/값 교환/XOR** 등 다양한 변형
3. **구조체 + 포인터 결합**: 단순 구조체가 아닌 **구조체 포인터 -> 연산자**, **2D 동적할당** 등 복합 문제 증가
4. **비트연산 신규 트렌드**: 2024년부터 비트시프트, 비트연산 + 구조체 결합 문제 등장

#### C언어 킬러 문제 패턴
```
[패턴 1] 포인터 + 문자열: 포인터 이동하며 문자열 조작
[패턴 2] 구조체 + 연결리스트: 노드 생성/연결/탐색/교환
[패턴 3] 2차원 배열/포인터: *(*(arr+i)+j) 형태의 접근
[패턴 4] 비트연산 + 조건: 삼항연산자와 비트시프트 결합
```

#### Java 출제 트렌드 핵심

1. **상속 + 다형성 = 최다빈출**: 9회 중 7회 이상 출제. **부모 타입으로 자식 참조 시 필드 vs 메서드 바인딩 차이**가 핵심
2. **재귀함수 꾸준**: 팩토리얼, 피보나치 변형, 중복제거 등 다양한 형태
3. **2025년 고급 문법 등장**: 람다식, enum, 제네릭 타입소거 등 **Java 8+ 문법** 출제 시작
4. **예외처리 2연속 출제**: 2024-3, 2025-1 연속 -> try-catch-finally 실행순서 반드시 숙지

#### Java 핵심 출제 공식
```
[공식 1] Parent p = new Child(); -> 메서드는 Child, 필드는 Parent
[공식 2] String 비교: == (주소) vs equals() (값)
[공식 3] static은 인스턴스 없이 호출 가능, 오버라이딩 불가
[공식 4] 생성자 체이닝: this() -> super() 순서
[공식 5] try-catch-finally: finally는 반드시 실행 (return 있어도)
```
